## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Select the Project:

`mv <src-selected-project> src`

Installation:

`bundle install`

To Start the Server:

`npm start`

To Visit the App:

`localhost:3000`
